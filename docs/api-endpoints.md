# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`
- `GET /api/session`

### Spots

- `GET /api/spots`
- `GET /api/spots/:id`

### Bookings

- `GET /api/bookings`
- `GET /api/bookings/:id`

### Reviews

- `GET /api/spots/reviews`
